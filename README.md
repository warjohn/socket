# Клиент-серверное взаимодействие

Это проект демонстрирует простое клиент-серверное взаимодействие с использованием сокетов в C++. Клиент отправляет файл на сервер, который затем сохраняет его на диске.

## Структура проекта

Проект состоит из двух основных файлов:

- `main_client.cpp`: Исходный код клиента, который считывает данные из файла и отправляет их на сервер.
- `main_server.cpp`: Исходный код сервера, который принимает данные от клиента и сохраняет их в новый файл.

## Описание файлов

### `main_client.cpp`

Этот файл содержит код для создания клиентского приложения. Основные функции:

1. **Создание сокета**: Создает сокет для TCP-соединения.
2. **Установка соединения**: Подключается к серверу на локальном хосте на порту 8080.
3. **Чтение файла**: Открывает файл `client_file.txt` для чтения в двоичном режиме.
4. **Отправка данных**: Считывает данные из файла в буфер и отправляет их на сервер.
5. **Закрытие сокета**: Закрывает сокет после завершения передачи данных.

### `main_server.cpp`

Этот файл содержит код для создания серверного приложения. Основные функции:

1. **Создание сокета**: Создает сокет для TCP-соединения.
2. **Привязка сокета**: Привязывает сокет к IP-адресу и порту 8080.
3. **Прослушивание входящих соединений**: Начинает прослушивание входящих соединений.
4. **Принятие соединения**: Принимает входящее соединение от клиента.
5. **Получение данных**: Считывает данные, отправленные клиентом, и записывает их в файл `received_file.txt`.
6. **Закрытие сокета**: Закрывает сокет после завершения передачи данных.
